## ⚙️ Setup

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/pirate-x.git
```

### 2. Open Project

Open the folder in VS Code.

### 3. Run Locally

Use Live Server extension
OR

```bash
python -m http.server 8000
```

Then open:

```bash
http://localhost:8000
```

---

## 🔑 TMDB API Setup

1. Create account on TMDB
2. Generate API key
3. Replace API key inside:

```js
script.js
watch.js
```

Example:

```js
const TMDB_KEY = "YOUR_API_KEY";
```
